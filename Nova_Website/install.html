Setup Guide

Get Nova running.

From zero to a running AI assistant in under 2 minutes. Step-by-step instructions for every OS.

Before You Start

Requirements.

🐍
Python 3.8+
python.org/downloads
📦
pip
Comes with Python
🌐
Chrome / Edge
For voice features
🔊
Speakers
For audio output
Installation

Step-by-step setup.

1
Download the project
Download

Download the ZIP file from GitHub and extract it to a folder on your computer.

terminal / command prompt 
# Option A: Clone with Git
git clone https://github.com/yourusername/Nova_Assistant.git
cd Nova_Assistant

# Option B: Download ZIP and extract, then open terminal in folder
cd Nova_Assistant
2
Check Python is installed
Verify

Open your terminal (or Command Prompt on Windows) and verify Python 3.8+ is installed.

Command Prompt / PowerShell 
python --version
# Expected output:
Python 3.11.0

# If "python" doesn't work, try:
python3 --version

If Python is not installed, download it from python.org/downloads. Make sure to check "Add Python to PATH" during installation on Windows.

3
Install required libraries
Install

In your terminal, inside the Nova_Assistant folder, run this one command to install everything:

terminal 
pip install -r requirements.txt

# This installs:
  flask       ← web server
  pyttsx3     ← text-to-speech (offline)
  wikipedia   ← Wikipedia search

If you get a permission error on Mac/Linux, use: pip3 install -r requirements.txt

4
Run the assistant
Launch 🚀

Start the Flask server with this command. Keep this terminal window open while using Nova.

terminal — inside Nova_Assistant/ 
python app.py

# You will see this output:
═══════════════════════════════════════════════════════
  ⚡  NOVA — AI Assistant is starting up...
═══════════════════════════════════════════════════════
  ➤  Open your browser: http://127.0.0.1:5000
  ➤  Press CTRL+C to stop the server
═══════════════════════════════════════════════════════
5
Open in your browser
Use it!

Open Google Chrome or Microsoft Edge (recommended for voice features) and go to:

browser address bar 
http://127.0.0.1:5000

✅ You should see the Nova chat interface. Type a message and press Enter — or click 🎤 to speak!

💡 Tip: Allow microphone access when prompted by the browser for voice input to work.

Editor Setup

Running in VS Code.

  1. Open VS Code and go to File → Open Folder → select Nova_Assistant
  2. Open the built-in terminal: View → Terminal (or press Ctrl + `)
  3. Install the Python extension by Microsoft if prompted
  4. In the terminal, run: pip install -r requirements.txt
  5. Then run: python app.py
  6. Click the link http://127.0.0.1:5000 that appears — or open it in Chrome manually
Free Hosting

Host your portfolio website.

The portfolio website (these pages) can be hosted free on GitHub Pages. The Python app runs locally — only the website can be hosted online.

💻 Run Locally (VS Code)
  • Open the Nova_Website/ folder in VS Code
  • Install the Live Server extension (by Ritwick Dey)
  • Right-click on index.html
  • Click "Open with Live Server"
  • Browser auto-opens at http://127.0.0.1:5500
  • Any file edits auto-refresh the browser ✅
🌍 Host Free on GitHub Pages
  • Create a free account at github.com
  • Create a new repository named nova-assistant
  • Upload all files from Nova_Website/ to the repo
  • Go to Settings → Pages in the repo
  • Under "Source" select Deploy from branch → main
  • Click Save. Your site is live at yourusername.github.io/nova-assistant ✅
Help

Troubleshooting common errors.

Error Cause Fix
ModuleNotFoundError: flask Flask not installed Run: pip install flask
ModuleNotFoundError: wikipedia Wikipedia library missing Run: pip install wikipedia
ModuleNotFoundError: pyttsx3 pyttsx3 not installed Run: pip install pyttsx3
Address already in use (port 5000) Another app uses port 5000 In app.py change port=5000 to port=5001, then go to :5001
No voice in browser Browser blocked audio/mic Click the 🔒 icon in address bar → allow Microphone and Sound
pyttsx3 no sound on Mac macOS TTS permission System Settings → Privacy → Accessibility → allow Terminal
python: command not found Python not in PATH Try python3 app.py or reinstall Python with "Add to PATH" checked
Mic button not working Browser unsupported Switch to Google Chrome or Microsoft Edge — Firefox does not support SpeechRecognition fully
