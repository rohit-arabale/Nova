All Features

Everything Nova can do.

From web searches to voice responses — here's the full capability list.

🔊
Voice Responses (TTS)
Every response is spoken aloud using two layers: pyttsx3 on the server (Python), and the browser's built-in Web Speech API. No internet needed for voice.
Offline, no API key needed Adjustable rate & volume
🎤
Voice Input (Mic)
Click the 🎤 button in the browser to speak your command. Nova uses the browser's SpeechRecognition API — no extra library needed.
Works in Chrome & Edge One tap to speak
📖
Wikipedia Search
Ask Nova to search Wikipedia and it will read out a 3-sentence summary. Handles disambiguation errors and missing pages gracefully.
search wikipedia Python wiki Albert Einstein
🔍
Google Web Search
Instantly opens a Google search in your browser. Just prefix with "search" or "find" — Nova builds the URL and launches it.
search web Python tutorials find best laptops 2026
🌐
Open Websites
Nova knows 18+ websites. Just say "open [name]" and it launches it in your default browser. Easy to add more in commands.py.
open youtube open github
🕐
Time & Date
Ask for the current time or today's date and Nova reads it back in a friendly format including the day of the week.
what time is it? what is today's date?
💬
Smart Conversation
Nova responds to greetings, "thank you", "bye", "who are you", and even tells jokes. Responses are randomized so it doesn't feel repetitive.
tell me a joke who are you?
⚡
Quick Command Chips
The web UI shows clickable shortcut buttons for the most common commands so you can try Nova immediately without typing.
One-click: Time, Date, Search One-click: YouTube, Help, Joke
🛡️
Error Handling
Every command is wrapped in try/except. If Wikipedia is down, pyttsx3 fails, or you type gibberish — Nova responds gracefully with a helpful message.
No crashes on bad input Friendly fallback messages
